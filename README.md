# quran-zh

古兰经 · 中文 — 一个简洁、易读的古兰经前端网站，提供阿拉伯原文及马坚（Ma Jian）中文译文。

## 功能特色

- **114 章完整列表** — 显示阿拉伯章名、中文译名、章节类型（麦加 / 麦地那）及经节数量
- **中文译文阅读** — 经文附阿拉伯原文与马坚中文译文对照显示
- **深色 / 浅色主题** — 一键切换，偏好自动保存
- **字体大小调节** — 可自由调整阿拉伯文字体大小
- **章节搜索** — 按编号、中文名或阿拉伯文快速过滤章节
- **翻页浏览** — 每页显示 10 节，支持前后翻页
- **响应式布局** — 适配手机、平板、桌面端

## 数据来源

- 经文数据：[Quran.com API v4](https://api.quran.com/api/v4)
- 中文译文：马坚（Ma Jian）译本，Translation ID 109

## 本地运行

```bash
# 任意静态文件服务器均可，例如：
python3 -m http.server 8080
# 然后在浏览器打开 http://localhost:8080
```
